# Task 1 - Enhanced Calculator (ShadowFox Java Internship)

## About
This project is an enhanced calculator built for the ShadowFox Java Development Internship.
It has two versions:
1. **Calculator.java** - Console-based version
2. **CalculatorGUI.java** - GUI version (Swing) with a dark theme, BODMAS expression parsing, and unit conversions

## Features
- Basic arithmetic: Addition, Subtraction, Multiplication, Division
- Scientific operations: Square Root, Power (Exponentiation)
- Unit conversions: Celsius to Fahrenheit, Fahrenheit to Celsius, USD to INR
- Percentage calculation
- Full expression support with BODMAS (e.g., 5+3*2 = 11)
- Uses BigDecimal for precise calculations (avoids floating point errors)
- Handles invalid input and divide-by-zero without crashing

## How to Run

### Console Version
1. Open terminal in the project folder
2. Compile: javac Calculator.java
3. Run: java Calculator

### GUI Version
1. Open terminal in the project folder
2. Compile: javac CalculatorGUI.java
3. Run: java CalculatorGUI

## Technologies Used
- Java (Core)
- Swing (GUI)
- BigDecimal (precision handling)
